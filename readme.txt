== Description ==

Use Wapuu as WordPress user avatar, support display in article comments, bbPress forum, BuddyPress profile, etc.

This plugin is improved based on [Wapuuvatar](https://wordpress.org/plugins/wapuuvatar/), from the original 25MB to the current 4 MB, including the newly added Wapuu avatar.

1. Added 333 wapuu avatars;

2. The high-definition pixel is 256px, and the minimum pixel is 64px;

3. The display is optimized, and the picture is adjusted to have a uniform length and width;

4. The speed is optimized, and the image is compressed to the greatest extent;


== Installation ==

* Download and install as any other WordPress plugin.
* Activate
* Head to Settings Â» Discussion
* Choose "Wapuu Avatar" under "Default avatar" - you can either have Wapuus everywhere, or Wapuu Local Avatar (Disable Gravatar).
* Enjoy all the Wapuus
